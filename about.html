TENTANG SAYA

Saya Arisatya Abhirama, disini Saya akan mendeskripsikan diri sendiri. Saya lahir di Surabaya pada tanggal 14 Juli 2003. Lalu Saya pindah ke Madiun semenjak Saya masih Taman Kanak-Kanak (TK). Sampai sekarang Saya tinggal di sebuah Perumahan yang letaknya di tengah Kota. Sekarang Saya kuliah di Politeknik Negeri Madiun dengan jurusan Teknologi Informasi. Sebelum kuliah, Saya pernah bekerja menjadi sales dan menjadi Karyawan Minimarket. Namun Saya hanya dapat bertahan selama 1 Bulan karena pekerjaan tersebut membuat Saya sering mengalami kelelahan. Hal tersebuut dapat membuat Saya terkena gangguan fisik bahkan sampai membuat Saya sedikit trauma. Saya memiliki banyak hobi, diantaranya suka main Game PS/Mobile, membaca Wattpad Fanfiction, main basket, dan juga Saya suka sekali pergi liburan bersama Keluarga Saya. Sekian dari deskripsi tentang Saya. Bilamana Saya memiliki banyak kesalahn mohon dimaafkan. TERIMA KASIH
